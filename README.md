# Mi-Web

Este es un proyecto de ejemplo para demostrar cómo crear una página web básica con HTML.

## Descripción

Mi-Web es una aplicación web sencilla que muestra diferentes elementos HTML incluyendo:
- Un mensaje de bienvenida
- Una tabla con datos de ejemplo
- Un formulario de contacto
- Párrafos de texto
- Imágenes de ejemplo
- Enlaces externos
- Un botón interactivo con JavaScript

## Características

✅ **HTML5 válido** con estructura completa (DOCTYPE, head, body)
✅ **Responsive** con meta viewport
✅ **Accesible** con etiquetas semánticas y atributos alt
✅ **Interactivo** con JavaScript inline para el botón

## Cómo usar

Simplemente abre el archivo `index.html` en tu navegador web favorito para ver la página en acción.

## Estructura del proyecto

```
Mi-Web/
├── README.md       # Este archivo
└── index.html      # Página web principal
```
